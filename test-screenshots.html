Sistema de Gamificação

Implementação completa de um sistema de gamificação para aumentar o engajamento dos usuários.

Principais recursos:

  • Perfis de Gamificação - Cada usuário possui um perfil com nível, pontos e conquistas
  • Sistema de Níveis - Progressão por Bronze, Prata, Ouro, Platina e Diamante
  • Conquistas - Desbloqueadas ao realizar ações específicas no sistema
  • Pontuação - Obtida ao completar atividades, avançar negociações e se comunicar
  • Ranking Semanal - Competição entre usuários com reset semanal

Características técnicas:

  • API completa para gerenciamento de gamificação
  • Rotas específicas para diretores visualizarem dados consolidados
  • Integração com o fluxo de trabalho existente
  • Sistema de notificações para conquistas

Conquistas disponíveis:

  • Velocista - Complete 5 atividades antes do prazo
  • Super Produtivo - Complete 10 atividades em um único dia
  • Negociador Master - Feche 3 negociações em uma semana
  • Comunicador - Mantenha comunicação diária por 7 dias
  • Consistência - Mantenha comunicação diária por 30 dias
  • Primeiro Lugar - Termine em primeiro no ranking semanal

API de Gamificação

Endpoints disponíveis:

  • GET /api/gamification/profile - Perfil do usuário atual
  • GET /api/gamification/points-history - Histórico de pontos
  • GET /api/gamification/achievements - Lista de conquistas
  • GET /api/gamification/weekly-ranking - Ranking semanal
  • GET /api/gamification/company-summary - Resumo da empresa (diretores)
  • GET /api/gamification/role-ranking - Ranking por função (diretores)
  • POST /api/gamification/test/add-points - Teste para adicionar pontos
